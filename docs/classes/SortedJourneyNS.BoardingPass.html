This class defines a basic skeleton of BoardingPass, its attributes are fully defined here. It is the basic unit which will be used in coming classes. Each boarding pass has a start_of_journey and end_of_journey in addition it contains information about seat assignment, gates and means of transportation (such as flight number, bus number etc). Some extra info is also stored such as Baggage collection.
